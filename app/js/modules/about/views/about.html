Hi! My name is Cristian Sergiu Drincu, but you can call me Chris.


I've been busy programming and designing apps since 2002. I've started by building my faculty's first multimedia CD, crafted with Macromedia Flash. After a TV application that I've built from scratch using Flash, I turned my eyes towards the front-end for the web: HTML, CSS and Javascript. Jquery followed soon after...

However, there was something missing. I wanted to build an entire app by myself, backend included. I first turned to PHP and later, while studying Computer Science, I discovered Java, which helped me build my first desktop application using Swing and JDBC. From there on out, the sky was the limit.

I consider myself to be a complete programmer, meaning that I can take a concept and implement it on both sides of the development cycle: frontend as well as the backend.

In 2007 I've founded my own company, Globe-Studios. This web project is a showcase of the work completed during 2007-2012, as well as a presentation of the team that developed the projects.

Currently I'm involved in building fullstack javascript applications using NodeJS, Express, MongoDB and AngularJS.
